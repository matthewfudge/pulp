Fonts

Drop a .ttf/.otf onto a row to bundle a non-system font into the export. Otherwise Pulp falls back to a system match or its bundled OFL set at import.

Click "Scan fonts" to detect fonts used in the current selection.

This plugin runs entirely on your machine. It does not send any data over the network. Source: pulp on GitHub.
